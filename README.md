Desarrolla una aplicación sencilla en Java para gestionar la información básica de los pilotos de Fórmula 1 de una temporada. Los datos se almacenan en un fichero de texto y se cargan en memoria para su tratamiento mediante colecciones.

Formato del fichero pilotos_f1.txt:
Cada línea representa un piloto con el siguiente formato:
nombre;escuderia;puntos

Métodos
1) Cargar pilotos
• Leer el fichero pilotos_f1.txt y cargar sus datos.
2) Mostrar pilotos
• Mostrar por pantalla todos los pilotos.
3) Añadir nuevo piloto
• Solicitar al usuario: nombre, escudería y puntos
• Añadir el piloto a la colección solo si no existe otro con el mismo nombre
4) Buscar piloto
• Solicitar el nombre de un piloto
• Mostrar toda su información
5) Guardar datos
• Guardar la lista completa de pilotos en el fichero pilotos_f1.txt.
• El fichero debe sobrescribirse con el contenido actualizado.
